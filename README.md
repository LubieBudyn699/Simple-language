# 🚀 Simple-language (v0.1-alpha)

A fast, human-readable programming language made for system automation and for people who hate complex syntax. It transpiles directly into highly optimized C++ code.

## 🛠️ How it works
Simple takes your clean, straightforward script, translates it line-by-line into clean C++, and compiles it using `g++` with aggressive optimizations (`-O3`).

## 💻 Code Example
Instead of writing complex boilerplates, in **Simple** you just write what you want to achieve:
